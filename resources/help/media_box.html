Media Box

Now let's look a little closer at the media box.

Here is where you can create and choose different media items for your services. Items include:

  • Songs: The lyrics for hymns and praise songs
  • Bible: Scripture passages to be shown during the service
  • Custom Slides: Slides with your own text
  • Images: Any images you may want to show during the service
  • Videos: Any videos you may want to show during the service
  • Web Slides: Internet content you may want to show during the service

Buttons

Each of these tabs has a set of buttons you can use to perform various tasks:

  • Create a new item
  • Add this item to the Order of service
  • Send the item directly to "live" (without adding it to the Order of Service

Additionally, if the tab has a search box, use to clear the search box.

Context Menu

Right-clicking on an item in the Media Box will bring up a menu where you can add that item to the order of service, remove the item from the program or, in the case of songs and custom slides, edit the item.

A note about scripture passages...

The best way to use the Bible tab of the media box is to import your favorite bible version(s) and add them to the service through the "Enter Passage" box. However, if you would prefer not to import a bible, or if you would like to show a passage from a version you rarely use, simply paste the text of that passage into the lower box of this tab. Ensure that your pasted text includes verse numbers or this will not work as expected.
