Menu Bar

At the top of the screen is a menu bar where you'll find many useful functions.

File Menu

In the file menu, you'll find options dealing with files and the program:

  • Create a New Service: Clear the Order of Service box to create a new service
  • Open a Service: Open a saved service file
  • Open Recent Service: Shows a list of services you have recently used
  • Save Service: Saves your current Order of Service to a file
  • Print Service: Prints your current Order of Service
  • Import: Shows a list of song import options
  • Export Songs: Export a song or songs in the OpenLyrics file format
  • Exit: Exit the program

Tools Menu

In the tools menu, you'll find various tools for working with the program:

  • Show/Hide Display Screen: Toggles hiding the display screen and showing the desktop
  • Show/Hide Black Screen: Toggles showing just a black screen
  • Show/Hide Logo Screen: Toggles showing a logo screen
  • Block Remote Inputs: Toggles blocking inputs from the ProjectOn remote
  • Theme: Here you can choose between a light (default) or dark theme for the program
  • Settings: Opens the Settings window
  • Import XML Bible: Imports a Zephaniah XML Bible into the program
  • Clear Song Database: Removes all songs from the program's database (useful if a wrong folder or file was imported)
  • Save/Change CCLI SongSelectPassword: Stores or changes your CCLI SongSelect login information

Help Menu

In the help menu, you'll find a couple of help options:

  • Help Contents: Shows this help window
  • About: Provides information about this program's version and licensing

Keyboard Shortcuts

Many of the Menu options have keyboard shortcuts associated with them. Use these shortcuts by holding down the special key(s) and pressing the associated letter:

  • CTRL-N: Create a New Service
  • CTRL-O: Open a Service
  • CTRL-S: Save Service
  • CTRL-P: Print Order of Service
  • CTRL-D: Show/Hide Display Screen
  • CTRL-B: Show/Hide Black Screen
  • CTRL-L: Show/Hide Logo Screen
  • CTRL-ALT-S: Open Settings Window
  • F1: Show this Help Window
  • CTRL-A: Show the About Window
