File Locations

Below are the locations of files you may want to find.

Data Files

The database, configuration file, images, videos, and bibles are stored in the program's directory (usually C:\Program Files\ProjectOn or C:\Program Files (x86)\ProjectOn). There you will find a folder called "_internal" in which is another folder called "data". In this data folder are multiple folders and files:

  • backgrounds: A folder containing all the slide background images you have imported
  • bibles: A folder containing any XML bibles you have imported
  • images: A folder containing any images you have imported
  • videos: A folder containing any videos you have imported
  • projecton.db: The main program database, in SQLite format
  • settings.json: The program's settings file, in JSON format

Device-Specific Settings

If the program has been installed on a shared drive or cloud folder, some settings are specific to the computer it is being run on. These are stored in a folder named "ProjectOn" under the "Roaming" Folder in the user's "AppData" folder. The file is named "localConfig.json" and is in the JSON format.

This is most easily found by opening Windows' File Explorer and typing into the navigation bar, "%appdata%\ProjectOn" (without the quotes).

Error Log

A log file is created by the program which tracks any errors that might occur while the program is running. Should you need to locate this file, it is found in the "ProjectOn" folder under the "Roaming" Folder in the user's "AppData" folder. It is named "error.log".
