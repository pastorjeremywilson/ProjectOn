Edit Window

Choosing to edit a Song or Custom Slide will bring up an editing window where you can make changes to that item's title, lyrics, text, etc.

Editor Elements

To the very top of the editing window you will find a title line. This will be the title of the song or custom slide.

If you are editing or creating a song, the next three lines will be for the Author, any copyright information associated with the song, and the number CCLI has assigned to this song (if the song comes from CCLI).

Again, if you are editing a song, the next part contains buttons where you can insert "tags" dividing the song up into verses, chorus, bridge, etc. Using tags allows you to, for example, type a chorus in just once, but display it multiple times while the song is sung.

Next is the lyric/text box. In here goes the lyrics for the song or the text you want your custom slide to say. Basic formatting functions are available for your text, namely Bold, Italic, and Underline. When using tags, all text underneath the tag belongs to that tag (up until the next tag or the end of the song). If you are not using tags, the program will automatically split the song wherever you have entered a blank line.

If you are editing a song, you can enter in the order you want verses, choruses, etc. displayed. For example, if your song has four verses with a chorus following each verse, you would type in v1 c1 v2 c1 v3 c1 v4 c1 . Separate each one with a space.

Advanced Options

Clicking the "+" beside "Show Advanced Options" will show additional options you can apply to your song or Custom Slide

By default, a song will display a footer showing the song's title, composer, any copyright info, and your CCLI license number, if you have one. To prevent this footer from showing, uncheck the box beside "Use Footer for this song."

Songs and custom slides will automatically use the default (global) font settings and the default (global) song background you have set in the "Settings" window or in the Toolbar. However, if you want to override those settings for this particular song or custom slide, check the box that says "Override Global Settings. Doing so will apply whatever font options or background options you set in this "Advanced Options" area. To retun to using the global settings for this song/custom slide, simply uncheck the "Override Global Settings" checkbox.

Finally, you have the "Save" and "Cancel" buttons. Press "Save" to save your changes, or press "Cancel" to undo any changes you have made. Using the "X" at the top-right of the window is the same as pressing "Cancel."
