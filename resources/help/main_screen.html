Main Screen

When you first open the program, you'll see the main screen. If you have a second monitor set up, you will also see the display screen.

Toolbar

Across the top is the toolbar.

From right to left, you have:

  • Save Button: to save your order of service
  • Open Button: to open a saved order of service
  • New Button: to create a new order of service
  • Settings Button: to open the program settings
  • Global Song Dropdown: selects the default background image for songs
  • Global Bible Dropdown: selects the default background image for bible passages
  • Show/Hide Display Screen: Hide or show the display screen, showing your desktop when hidden
  • Blank Screen: Show an entirely black display screen
  • Logo Screen: Show your own logo image on the display screen

Order of Service

To the left, you have the order of service and media boxes.

The order of service box is where you'll place your songs, scripture passages, etc., in the order they occur in your service. The order can be rearranged by dragging each item, or by using the arrows on the right.

The media box is where your songs, bible, custom slides, images, and videos are stored. Drag, double-click, or click the icon to add an item to the order of service.

Preview Pane

In the center of the screen, you have the preview box. This shows you how the currently selected order of service item will look when it goes "live."

If an item has multiple slides - for example, a song or scripture passage - they will be listed in the upper portion of this box. In the bottom is a small preview of how the slide will look. Pressing space when any of the preview items are selected will send that item to "live."

Live Pane

To the right, you have the live box.

Whatever item is currently selected in this box will show on the display screen. Clicking on an item will make that item show on the display screen, and the arrow keys of the keyboard will also change what is being displayed:

  • Up Arrow: Show the previous item
  • Down Arrow: Show the next item
  • Left Arrow: Show the previous item in the Order of Service
  • Right Arrow: Show the next item in the Order of Service

Pressing the Down Arrow on the last item in this box will automatically show the next item of the Order of service. Likewise, pressing the Up Arrow on the first item in this box will automatically show the previous item of the order of service. This allows you to seamlessly navigate the entire order of service from this box.

Note that the "live" box will also accept input from a PowerPoint remote control. Previous and Next slide buttons will perform the same functions as the up and down arrows. You should also be aware that this box needs to be "active" (have an item selected) in order for the PowerPoint remote to work.
