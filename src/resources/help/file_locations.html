File Locations

Below are the locations of files you may want to find.

Data Files

The database, configuration file, images, videos, and bibles are stored, by default, in a folder named "ProjectOn" in the "Roaming" folder of your user's AppData folder (C:\Users\your_user_name\AppData\Roaming\ProjectOn). In this data folder are multiple folders and files:

  • backgrounds: A folder containing all the slide background images you have imported
  • bibles: A folder containing any XML bibles you have imported
  • images: A folder containing any images you have imported
  • videos: A folder containing any videos you have imported
  • projecton.db: The main program database, in SQLite format
  • settings.json: The program's settings file, in JSON format

This is most easily found by opening Windows' File Explorer and typing into the navigation bar, "%appdata%\ProjectOn" (without the quotes).

Device-Specific Settings

If the program has been installed on a shared drive or cloud folder, some settings are specific to the computer it is being run on. These are stored in a file in the above-named data folder. The file is named "localConfig.json" and is in the JSON format.

Error Log

A log file is created by the program which tracks any errors that might occur while the program is running. Should you need to locate this file, it is found in the above-named data folder. It is named "error.log".
