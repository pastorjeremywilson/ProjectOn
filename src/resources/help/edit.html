Edit Window

Choosing to edit a Song or Custom Slide will bring up an editing window where you can make changes to that item's title, lyrics, text, etc.

Song Editor Elements

To the very top of the editing window you will find a Title line. This will be the title of your song. The next three lines will be for the Author, any Copyright information associated with the song, and the number CCLI has assigned to this song (if the song comes from CCLI).

Next is the lyrics box. In here are the lyric blocks of your song. Lyric blocks are the individual parts of your song. For example, Verse 1, Verse 2, Chorus 1, and Ending 1. You can add a block of lyrics by clicking the Add Lyrics button. Doing so will create a new lyrics block where you can specify the type of lyrics (Verse, Chorus, Bridge, etc.), the number, (Verse 1, Verse 2, etc.), and the lyrics themselves. Basic formatting functions are available for your text, namely Bold, Italic, and Underline.

It might be more convenient to edit all the lyrics at once. For example, if you want to paste all the lyrics from your computer's clipboard. To do so, click the Edit All Lyrics toggle button. The editor changes to a view that shows all lyrics you currently have, as well as a row of buttons you can use to insert the tags that will identify what each part of your lyrics are (Verse, Chorus, etc.). Whatever text is below the tag you insert belongs to that tag. Once you are satisfied with how you've edited the lyrics, click the Edit All Lyrics toggle button again and the lyrics box will divide up your lyrics according to your inserted tags.

Your next step will be to specify the order in which you want verses, choruses, etc. to be displayed. To the right of the lyrics, you will see a box labeled Song Order. Simply drag-and-drop the lyric blocks you have created in their correct order in the Song Order box. You can rearrange any of the items in the Song Order box by dragging them up and down, and if you need to remove a segment from the song's order, click on it then press the Delete button on your keyboard.

Custom Slide Settings

Clicking the "+" beside Show Advanced Options will show additional options you can apply to your song or Custom Slide

By default, a song will display a footer showing the song's title, composer, any copyright info, and your CCLI license number, if you have one. To prevent this footer from showing, uncheck the box beside "Use Footer for this song."

Songs and custom slides will automatically use the default (global) font settings and the default (global) song background you have set in the "Settings" window or in the Toolbar. However, if you want to override those settings for this particular song or custom slide, check the box that says Override Global Settings. Doing so will apply - to this song or custom slide only - whatever font options or background options you set in this "Advanced Options" area. To retun to using the global settings for this song/custom slide, simply uncheck the "Override Global Settings" checkbox.

Finally, you have the Save and Cancel buttons. Press "Save" to save your changes, or press "Cancel" to undo any changes you have made. Using the "X" at the top-right of the window is the same as pressing "Cancel."
